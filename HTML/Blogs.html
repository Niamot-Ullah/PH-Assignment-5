Question-1: What are the different ways to select an element in the DOM?

We can select elements in different ways, such as : getElementById, getElementsByClassName, getElementsByTagName, querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference among them is, if we use innerHTML we will get the HTML content inside an element. if we use innerText , we get visible text inside an element. if we use textContent, we get the full text content of an element, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a pattern in JavaScript where instead of adding event listeners to multiple child elements, you add a single event listener to a parent element and use event bubbling to catch events on its children.

Question-4: What is event bubbling in the DOM?

When an event triggered on a child element propagates to its parent and then further up the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Firstly, if i create, i will use document.createElement('tagName') , secondly , if i add , i will use document.body.appendChild(tagName), thirdly , if i remove , i will use, tagName.remove() or parent.removeChild(child)
